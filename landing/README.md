# Phantom Tracks

Phantom Tracks is a steganography experiment that turns Spotify playlists into a quiet filesystem and Spotify DMs into a private courier for covert data transmission.

## Project Overview

Hide and share data inside Spotify playlists. Data is encoded into ordinary-looking playlists, hosted by Spotify, with no infrastructure required. Share covertly through Spotify DMs—the carrier is a song, the payload is yours.

## Features

- **Hidden in plain sight** - Data encoded into ordinary-looking playlists
- **Zero infrastructure** - Spotify hosts the bytes, you hold the key
- **Built for whispers** - Share covertly through Spotify DMs
